## userdb の場所が移動した

`/etc` → `/etc/authlib`

## courier-authlib の起動が必要

```sh
/sbin/service courier-authlib start
```

するか

```sh
/sbin/chkconfig --level 2345 courier-authlib on
```

して

```sh
reboot
```

## makeuserdb の問題

```sh
$makedat="/usr/libexec/courier-authlib/makedatprog";
```

となっているが，`makedatprog` は `/usr/libexec/courier-imap` に入っていた．したがって `makedatprog` の場所を移動．
